# diskfetch

diskfetch - it's like neofetch, but for your disks. 
Diskfetch is terminal user interfase util for linux,
for get more info of your disks.

# Syntax
diskfetch <flags> <path to your disk>

flags:
    -h      - outputs this information
    -v      - outputs version
